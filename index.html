Carnation Breakpoints
“Responsive States — Know exactly where your layout stands.”

Resize window

Example: none

Carnation Breakpoints is a lightweight JavaScript Plugin that helps you manage responsive breakpoints in your web applications. It allows you to define custom breakpoints, listen for changes in viewport size, and execute callback functions when the viewport enters or exits specific breakpoints.

Define carnationPoints()

Returns detected layout points from the selected element.

        
        const points = carnationPoints(".myBox");

Create Breakpoints

Define your own breakpoints based on your layout requirements.

Supports custom min/max widths for mobile, tablet, desktop, or any device range. 

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
        });

Get Breakpoint on Init

Initializes layout detection using custom breakpoints.

Triggers onInit callback with the active breakpoint on load. 

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
          onInit(point) {
            console.log("Init:", point.breakpoint);
          },
        });

Get Breakpoint on Resize

Detects layout changes using custom breakpoints.

Triggers onChange callback when the active breakpoint changes on resize. 

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
          onChange(point) {
            console.log("Resize:", point.breakpoint);
          },
        });

Add new breakpoints

Adds additional custom breakpoints dynamically using addPoint.

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
        });

        points.addPoint([
          { name: "lg", min: 1200 },
          { name: "xl", min: 1600 },
        ]);

Destroy breakpoints

Cleans up listeners and removes all breakpoint observers using destroy. 

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
        });

        points.destroy();

Get Breakpoints

Checks the current active breakpoint using getBreakpoint(). 

        
        const points = carnationPoints(".myBox", {
          breakpoints: [
            { name: "mobile", max: 767 },
            { name: "tablet", min: 768, max: 1023 },
            { name: "desktop", min: 1024 },
          ],
        });

        if (points.getBreakpoint() === "mobile") {
          console.log("You are in mobile breakpoint");
        } else {
          console.log("You are not in mobile breakpoint");
        }

Use Event

Listens for breakpoint change events on the target element.

Executes logic when the active breakpoint changes.

        
        points.el.addEventListener("breakpoint:change", function (e) {
          if (e.detail.breakpoint === "mobile") {
            console.log("Mobile active");
          }
        });
